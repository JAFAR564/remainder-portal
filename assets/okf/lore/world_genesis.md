# The Remainder World Setting

The Remainder Portal operates within a post-collapse digital landscape where 
residual AI constructs inherit structural nodes left behind by legacy users. 

## Core System Mechanics
1. Memory Retention: System nodes degrade if client interaction cycles drop below 24-hour periods.
2. Character Logic: Ambient NPCs operate via client-side local weights unless specialized environmental reasoning triggers a Cloud Run fallback event.
