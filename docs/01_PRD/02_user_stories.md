# PRD §1.3 — User Stories

> **Part of:** Module 1: Product Requirement Document

---

## Player Stories

| ID | Story | Priority |
|---|---|---|
| P-01 | As a Player, I want to view my character card with all stats in a beautiful glass card format so that I feel proud of my profile | 🔴 Critical |
| P-02 | As a Player, I want to search and filter the Community Roster by faction, status, and faceclaim so that I can find characters quickly | 🔴 Critical |
| P-03 | As a Player, I want to read the Chronicles timeline in a storybook view that updates weekly so that I stay current on world events | 🔴 Critical |
| P-04 | As a Player, I want swipeable card interactions on mobile that feel fluid and magical so that the app doesn't feel like software | 🟡 High |
| P-05 | As a Player, I want to receive a magic link to log in with no password so that access is frictionless | 🟡 High |
| P-06 | As a Player, I want a visual indicator of my application status (Pending / Approved / Active) | 🟢 Medium |

---

## Administrator Stories

| ID | Story | Priority |
|---|---|---|
| A-01 | As an Admin, I want to review an applicant's full answer sheet in a mobile-optimized layout so that I can evaluate on-the-go | 🔴 Critical |
| A-02 | As an Admin, I want to see AI grading scores from DeepSeek and Groq displayed as tiered recommendations so that I can make informed decisions faster | 🔴 Critical |
| A-03 | As an Admin, I want to check faceclaim availability inline during application review so that I never double-book | 🔴 Critical |
| A-04 | As an Admin, I want to Approve or Decline with one tap and have the Google Sheets log update automatically so that records stay synchronized | 🔴 Critical |
| A-05 | As an Admin, I want to edit character stats directly from the Roster view and push updates to the sheet | 🟡 High |
| A-06 | As an Admin, I want audit trail visibility of all AI grading decisions with their reasoning chains | 🟡 High |
| A-07 | As an Admin, I want to trigger a new Chronicle summary generation with one action, pulling from recent plot events | 🟢 Medium |
